## 📖 About

This project is a RESTful API for managing movies.  
It uses Entity Framework Core, MSSQL Server, GraphQL, Redis.

[Open API Swagger](https://localhost:5001/swagger)

---

Clone the repository:

```bash
git clone https://github.com/TaherJsStack/Building_MinimalAPIsMoviesApp.git


Navigate into the project:

```bash
cd Building_MinimalAPIsMoviesApp

Install dependencies:

```bash
dotnet restore


## 🛠 Tech Stack

- **.NET**: .NET 8 
- **Framework**: ASP.NET Core Web API
- **ORM**: Entity Framework Core
- **Database**: MSSQL Server 
- **Authentication**: JWT 
- **Tools**: Swagger
